# SmallPot_Robot
小锅机器人，微信工作群助手，发送关键字触发机器人查询，机器人回复需要的内容
主要模块

- itchat
- matplotlib
- pymssql
- prettytable
- PIL

查询语句
查询进度={brand}
查询出货={begin_tiem}.{end_time}
查询产量={begin_tiem}.{end_time}.{type}
拣货差异={begin_tiem}.{end_time}
条码库存={item}
查询退供={order_id}
未收汇总={days} 
待上架汇总={days}
